# MEA

Matrix Exponential Attention

$$
\mathrm{MExp}(\mathbf{Q} \mathbf{K}^{\top}) \mathbf{V} 
$$

See Higher-order Linear Attention (HLA) (https://arxiv.org/abs/2510.27258) 

MEA can be implemented using taylor expansion of the matrix exponential based on HLA.
